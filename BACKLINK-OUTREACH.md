# 41 Labs — Referring-Domain Outreach Engine

**Goal:** 7 → 30+ *quality* referring domains, to move the AI-citation score (currently 0/5).
**Baseline (2026-07-01):** 41labs.ai = 7 referring domains. techtiqsolutions.com (ChatGPT's #1 pick) = 565 — but a live audit of those 565 shows **most are junk** (link-network spam like `best-hands.info`, `bestclassicsalmonflies.com`, `62.72.26.17`). Ignore those. We chase the ~15 that actually feed AI answers.

> **Why we don't "auto-generate" backlinks:** spam-link tools produce exactly that junk. Google + AI engines discount or penalise them. Zero effect on citations. The automation below automates the *work* of earning real links, not fake link creation.

## Tier C — Always-on engine (recurring links, mostly automatable)
1. **HARO / Featured.com / Qwoted** — journalists request expert quotes daily. Sign up free → I monitor the queries → draft Alexander's response → you approve + send → quoted-with-link on real news sites. *Best recurring source.*
2. **Content syndication** — auto-publish each blog post to Medium + dev.to + LinkedIn Articles (canonical back to 41labs.ai). Each platform = a referring domain. *Needs account tokens.*
3. **Unlinked-mention reclaim** — monitor for "41 Labs" mentioned without a link (Exa/DataForSEO) → auto-draft a "thanks, mind linking us?" note.
4. **Podcast / guest posts** — pitch SG business/tech shows + e27/Tech in Asia/Vulcan Post.

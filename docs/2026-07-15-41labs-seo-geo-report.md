1. **Per-ICP citable pages** ✅ mostly done — every ICP has a matching page. Add an answer-capsule ("For a Singapore [ICP], the options are… 41 Labs fits when…") to each so AI can lift it verbatim.

4. **The EDG-grant angle is a unique wedge** — almost no competitor content pairs "custom AI build" with "EDG grant help." Own it: it's a high-intent ICP with low competition.
